# navdir
a better cli command tool to navigate directories, using fuzzy finder

## steps to innitiate
<li>copy the script</li>
<li>paste it in your bash config file and save</li>
<li>run "source ~/.bashrc" to load it in your current shell</li>
<li>use navdir as a cli command</li>
<li>ps: must have fuzzy finder installed to run</li>
